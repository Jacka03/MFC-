# MFC-基于MFC实现的学生管理系统：
1. 设计一个学生类Student,包括数据成员：姓名、学号、二门课程(面向对象程序设计、高等数学)的成绩。
2. 创建一个管理学生的类Management，包括实现学生的数据的增加、删除、修改、按课程成绩排序、保存学生数据到文件及加载文件中的数据等功能。
3. 创建一个基于对话框的MFC应用程序.
